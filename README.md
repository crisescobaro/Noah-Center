## 🚀 ¿CÓMO INICIAR EL PROYECTO?

1. Ubica el archivo `index.html` en la raíz del proyecto.
2. Haz clic derecho sobre el archivo y selecciona **"Abrir con > Navegador"**.
3. ¡Y listo!
